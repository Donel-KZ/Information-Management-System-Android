# Information Management System (Android App)

# Overview
This Android application was developed for Central University to manage student information efficiently. It allows staff to perform CRUD operations (Create, Read, Update, Delete) using an SQLite database.

##  Features
- Register new students
- View all student records
- Update student information
- Delete student records
- SQLite database integration
- Toast messages for feedback

## Technologies Used
- Java
- Kotlin
- Android Studio
- SQLite


## Database Structure
Table: Students
- ID (Primary Key)
- Name
- Department
- Phone Number

## Learning Outcome
- Android UI Design
- SQLite database handling
- CRUD operations
- Data persistence
